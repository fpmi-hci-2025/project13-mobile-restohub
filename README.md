# RestoHUB Mobile - iOS приложение для доставки еды

## Обзор продукта
RestoHUB Mobile — нативное iOS-приложение для заказа еды и поиска ресторанов, предлагающее пользователям максимально удобный и быстрый способ найти идеальное блюдо с доставкой прямо к двери. Приложение сочетает в себе передовые технологии Apple с уникальным пользовательским опытом.

## Основные функции и возможности

### Поиск и открытие ресторанов
- **Интеллектуальные рекомендации** — персонализированные предложения на основе истории заказов и предпочтений
- **Продвинутая фильтрация** — поиск по типу кухни, ценовому диапазону, рейтингу, времени доставки и диетическим требованиям
- **Карта с ресторанами** — визуальное отображение ресторанов с зонами доставки и текущей загрузкой
- **Коллекции и подборки** — тематические подборки: новые рестораны, быстрая доставка, выбор шефов, локальные хиты

### Корзина и оформление заказа
- **Автоматическая группировка** — разделение заказов из разных ресторанов с учетом минимальных сумм
- **Совместные покупки** — функция групповых заказов с автоматическим разделением счета
- **Быстрый повтор** — мгновенный повтор прошлых успешных заказов
- **Мониторинг цен** — уведомления об изменениях цен в любимых заведениях

### Пользовательский интерфейс
- **Динамичное меню** — интерактивные анимации, параллакс-скроллинг, микро-интеракции
- **Экспресс-режим** — упрощенный интерфейс для быстрых заказов
- **Сезонный дизайн** — адаптация цветовых схем и иллюстраций к времени года и праздникам
- **Элементы геймификации** — система достижений, бейджей и наград за активность

### Отслеживание доставки
- **Реал-тайм трекинг** — точное отслеживание курьера на карте в реальном времени
- **Умный ETA** — динамический расчет времени доставки с учетом множества факторов
- **Мультитрекинг** — одновременное отслеживание нескольких активных заказов

### Управление профилем
- **Умные адреса** — система запоминания и автоматического предложения частых адресови

### Технологический стек
- **Платформа:** iOS 15.0+
- **Язык:** Swift 5+
- **UI Фреймворк:** SwiftUI с элементами UIKit
- **Архитектура:** MVVM + Coordinator
- **Управление состоянием:** Combine
- **Сетевое взаимодействие:** URLSession + Async/Await
- **Локальное хранение:** Core Data + UserDefaults
- **Зависимости:** Swift Package Manager
- **Карты и геолокация:** MapKit + CoreLocation
- **Изображения:** AsyncImage с кастомным кэшем
- **Аналитика:** Firebase

### Ключевые архитектурные принципы
- **Модульность** — каждый функциональный блок независим и тестируем
- **Протокольно-ориентированный дизайн** — абстракция через протоколы
- **Реактивное программирование** — Combine для управления потоками данных
- **Внедрение зависимостей** — упрощение тестирования и поддержки
- **Конфигурация окружения** — разные настройки для dev/staging/production

## Информация для разработчиков

### Быстрый старт разработки
##### Клонирование репозитория
```bash
git clone https://github.com/fpmi-hci-2025/project13-mobile-restohub.git
```

##### Переход в директорию проекта
```bash
cd project13-mobile-restohub
```

##### Открытие проекта в Xcode
```bash
open RestoHUB.xcodeproj
```

#### Выбор симулятора и запуск
- 1. Выбрать схему "RestoHUB"
- 2. Выбрать симулятор iPhone 15+
- 3. Нажать Cmd + R для запуска
